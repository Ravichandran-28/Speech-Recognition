# 🎙️ Speech Emotion Recognition Using Deep Learning

A Deep Learning-based Speech Emotion Recognition (SER) system that identifies human emotions from speech signals using Convolutional Neural Networks (CNN). The project extracts audio features such as MFCCs and classifies emotions like Happy, Sad, Angry, Fear, Neutral, Disgust, and Surprise to improve Human-Computer Interaction (HCI). Developed using Python, TensorFlow, Keras, and Librosa. :contentReference[oaicite:0]{index=0}
